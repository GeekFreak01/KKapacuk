# Schedule App

Android приложение на Kotlin + Jetpack Compose для отслеживания текущей пары.

## Запуск

1. Откройте проект в Android Studio.
2. Дождитесь синхронизации Gradle.
3. Запустите приложение на эмуляторе или устройстве.

## Возможности MVP

- Определение сезона расписания по месяцу.
- Определение типа дня (будний / суббота).
- Список пар с прогрессом, подсветкой текущей и маркером половины.
